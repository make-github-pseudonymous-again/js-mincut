:haircut_woman: [@aureooms/js-mincut](https://aureooms.github.io/js-mincut)
==

Minimum cut algorithm for JavaScript.
See [docs](https://aureooms.github.io/js-mincut/index.html).

```js
// 5      2
// |\    /|
// | 0--1 |
// |/    \|
// 4      3
mincut([[0,1],[1,2],[1,3],[2,3],[0,4],[0,5],[4,5]]); // 0--1
```
